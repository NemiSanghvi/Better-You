# Better You

**Better You** is a calm, offline-first **AI-powered self-improvement app** built with React Native (Expo).  
It helps users focus on improving **one meaningful thing** by delivering **one task per day**, without overwhelm, feeds, or distractions.

The app is intentionally minimal and designed around consistency, clarity, and respectful accountability.
